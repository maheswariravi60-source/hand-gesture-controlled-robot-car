# Hand Gesture Controlled Robot Car

## Project Overview

Developed a hand gesture controlled robot car using Arduino and accelerometer-based motion sensing. The system allows wireless control of the robot car through hand movements, eliminating the need for traditional remote controllers.

## Technologies Used

- Arduino Uno
- Accelerometer Sensor (ADXL335/MPU6050)
- RF Module (433 MHz / nRF24L01)
- Motor Driver (L293D/L298N)
- DC Motors
- Embedded C

## Features

- Wireless gesture-based control
- Forward, backward, left, and right movement
- Real-time response to hand movements
- RF communication between transmitter and receiver

## Working Principle

The accelerometer detects hand movements and sends the data to the Arduino transmitter. The transmitter communicates wirelessly with the receiver module attached to the robot car. Based on the received data, the robot car performs the corresponding movement.

## Applications

- Robotics
- Industrial Automation
- Assistive Technologies
- Remote Vehicle Control

## Outcome

Successfully implemented a gesture-controlled robotic vehicle demonstrating wireless communication, sensor integration, and embedded system design.
